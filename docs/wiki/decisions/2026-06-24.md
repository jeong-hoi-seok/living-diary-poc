## 결정 | Expo SDK 54.0.35 정확 패치 핀
- 채택: `expo@54.0.35` (마이너 아닌 정확 패치 고정).
- 이유: 공유 위키 결정 [[Expo SDK 54 버전 고정]]을 그대로 따름.
  - 스토어 Expo Go가 지원하는 최신 SDK = 54 → 추가 빌드·멤버십 없이 실기기 즉시 테스트.
  - SDK 54(RN 0.81) = 레거시 아키텍처 지원 마지막 SDK.
  - 정확 패치 핀 → 팀·CI 재현성.
- 스캐폴드 방법: `create-expo-app --template expo-template-default@sdk-54` (템플릿 54.0.61)로 생성 후, `expo@54.0.35` 강제 + `expo install --fix`로 전체 의존성을 해당 패치 known-good 세트에 정렬.
- 트레이드오프: 위키 명시값 54.0.35로 고정 → 현재 더 최신 54 패치(54.0.62)의 패치 픽스는 미반영. 올릴지는 사용자 판단.
